# 准备工作
## 环境依赖
### node.js
安装最新版本的node
* [官网](https://nodejs.org/en/)

### yarn
在cmd中安装包管理工具
```bash
$ npm install -g yarn
```

### nrm
node库依赖源管理工具
```bash
$ yarn global add nrm
